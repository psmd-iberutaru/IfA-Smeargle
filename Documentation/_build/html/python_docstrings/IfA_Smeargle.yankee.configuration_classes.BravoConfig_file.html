IfA_Smeargle.yankee.configuration_classes.BravoConfig_file module¶

class IfA_Smeargle.yankee.configuration_classes.BravoConfig_file.BravoConfig(file_name=None)¶

Bases: IfA_Smeargle.yankee.configuration_classes.BaseConfig_file.BaseConfig

This is the configuration class of the BRAVO line.

The BRAVO line is mostly responsible for gathering the data from the raw output of the detector and processing it in a form that the later lines can use.

By default, all of the entries in the configurations are empty. Moreover, each configuration attribute only contains the required entries. Optional entries may be added at user discretion (see documentation for such entries).

detector_name¶

The name of the detector that is being processed.
