IfA_Smeargle.yankee.configuration_classes.YankeeConfig_file module¶

class IfA_Smeargle.yankee.configuration_classes.YankeeConfig_file.YankeeConfig(file_name=None)¶

Bases: IfA_Smeargle.yankee.configuration_classes.BaseConfig_file.BaseConfig

This is the configuration class of the YANKEE line.

The YANKEE line is mostly responsible for holding, storing, and reading of configuration files and classes that dictate the rest of the module’s behavior.

By default, all of the entries in the configurations are empty. Moreover, each configuration attribute only contains the required entries. Optional entries may be added at user discretion (see documentation for such entries).
