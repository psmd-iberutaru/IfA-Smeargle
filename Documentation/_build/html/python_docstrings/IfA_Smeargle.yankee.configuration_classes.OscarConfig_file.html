IfA_Smeargle.yankee.configuration_classes.OscarConfig_file module¶

class IfA_Smeargle.yankee.configuration_classes.OscarConfig_file.OscarConfig(file_name=None)¶

Bases: IfA_Smeargle.yankee.configuration_classes.BaseConfig_file.BaseConfig

This is the configuration class of the OSCAR line.

The OSCAR line is mostly responsible for creating good histograms and heat-maps of pixels and their value for further analysis.

By default, all of the entries in the configurations are empty. Moreover, each configuration attribute only contains the required entries. Optional entries may be added at user discretion (see documentation for such entries).
