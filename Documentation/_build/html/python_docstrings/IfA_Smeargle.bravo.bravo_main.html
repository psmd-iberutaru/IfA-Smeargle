IfA_Smeargle.bravo.bravo_main module¶

Contains all of the methods used for renaming and reformatting data.

Each function is dependent on the array and/or method. Only one function should ever be used for any one given process.

IfA_Smeargle.bravo.bravo_main.bravo_execution_saphria(data_directory, configuration_class)[source]¶

Reformats data provided by a SAPHRIA array into a form used by this program.

Each array or data set has their own different forms of representing the data that has been collected. The purpose of this, and other modules, is to reformat and restructure the data collected into a more useful form for this program.

The function automatically reformats the directories.

Parameters

  • data_directory (ndarray) – The data array that is to processed and filtered accordingly

  • configuration_class (SmeargleConfig or BravoConfig class) – The configuration class that will be used to provide instruction to the ECHO filters.

Returns

Return type

nothing
