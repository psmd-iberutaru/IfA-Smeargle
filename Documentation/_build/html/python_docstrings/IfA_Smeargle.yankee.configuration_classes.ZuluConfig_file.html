IfA_Smeargle.yankee.configuration_classes.ZuluConfig_file module¶

class IfA_Smeargle.yankee.configuration_classes.ZuluConfig_file.ZuluConfig(file_name=None)[source]¶

Bases: IfA_Smeargle.yankee.configuration_classes.BaseConfig_file.BaseConfig

This is the configuration class of the ZULU line.

The ZULU line is responsible for collecting all functions for each detector analysis line and grouping them into one method.

By default, all of the entries in the configurations are empty. Moreover, each configuration attribute only contains the required entries. Optional entries may be added at user discretion (see documentation for such entries).

Note

All built-in functions of the configuration classes are inherited from the BaseConfig class.

Configuration classes such as this one is generally wrapped within the SmeargleConfig class.
